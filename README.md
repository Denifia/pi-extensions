# pi-extensions

Personal extensions for the [Pi coding agent](https://github.com/badlogic/pi-mono).

## Extensions

| Extension | What it does |
|---|---|
| [`pi-copilot-multiplier`](./pi-copilot-multiplier) | Show GitHub Copilot premium request multipliers for models. |
| [`pi-delegate`](./pi-delegate) | Adds a `/delegate` command that summarises the current pi conversation and creates a GitHub issue for GitHub Copilot Coding Agent. |
| [`pi-windows-bash-guard`](./pi-windows-bash-guard) | Prevents Windows-specific cmd.exe habits from leaking into pi's bash tool and rewrites common `nul` usages safely. |
